明无梦的 Vim 使用记录

快捷键

2014-00-00

查看快捷键帮助

有两种情况：

  1. vim 默认的快捷键，可通过 :help c-r 类似的命令查看
  2. 通过 map 定义的快捷键，可通过 :map ,jd 类似的命令查看

自定义

  • <Leader>m2 查看光标所在单词的 man 2 手册
  • <Leader>m3 查看光标所在单词的 man 3 手册
  • <Leader>p 切换粘贴和普通模式
  • <Leader>/ 关闭搜索高亮
  • <Leader>a 到行首
  • nmap <Leader>e 到行尾
  • nmap <Leader>pa 在结对符之间跳转，助记 pair
  • <Leader>man 查看光标所在单词的 man 手册
  • <tab> 跳转到下一个 buffer
  • <c-tab> 跳转到上一个 buffer

插件快捷键

Vundle

  • :VundleInstall 安装插件
  • :VundleUpdate 更新插件
  • :Vundleclean 清除不用的插件

YouCompleteMe

  • <C-a> 主动调用补全
  • <leader>jd 跳转到定义或声明处

Syntastic

  • <leader>d 在 vim 底部显示详细错误描述

NERD Commenter

  • <leader>cc 注释当前行
  • <leader>cm 只用一组符号来注释
  • <leader>cy 注释并复制
  • <leader>cs 优美的注释
  • <leader>cu 取消注释

vim-easymotion

  • ff 全屏搜索
  • fs 往下搜索
  • fF 往上搜索
  • fl 行内向右搜索
  • fj 行间向下搜索
  • fk 行间向上搜索
  • fh 行内向左搜索
  • fw 往下搜索一个单词开始处
  • fb 往上搜索一个单词开始处
  • fe 往下搜索一个单词结尾处
  • fge 往下搜索一个单词结尾处

UltiSnips

  • <c-j> 扩展片段
  • <c-j> 往下跳转
  • <c-k> 网上跳转

CtrlP

  • <c-p> 调用查找
  • <F5> 刷新，在当前目录下重建缓存
  • <c-y 在当前路径下创建新文件
  • <c-j>, <c-k> 在搜索列表中上下移动

查找

  • f{char} 查找下一处指定字符出现的位置，如果找到了，就把光标移动到那里
  • ; 重复查找上次 f 命令所查找的字符

编辑

  • . 重复上次修改，每次进入插入模式也会形成一次修改（移动光标会重置修改状态）

插入

  • a 在当前光标之后添加内容

  • A = $a 在当前行的结尾添加内容

  • C = c$

  • s = cl
  • S = ^c
  • I = ^i
  • A = $a
  • o = A<CR>
  • O = ko

选中括号中的内容

  • vi', vi", vi( (or vib), vi[, vi{ (or viB), vi< 分别选中配对括号中的内容，不包括括号
  • 将以上的 v 换成 c 就是替换，d 是删除，y 是复制
  • 将以上的 i 换成 a 就是选中括号内及括号的内容

缩进

  • >> 增加当前行的缩进层级
  • >G 增加从当前行到文档末尾的缩进层级
  • >gg 增加从当前行到文档开始的缩进层级
  • < 用法与 > 类似

折叠

:help folding

  • zR 打开所有折叠
  • zM 关闭所有折叠
  • za 打开或关闭光折叠, zA 循环打开或关闭折叠
  • zi 翻转 'foldenable' 的值
  • zj 移动光标到下一个折叠开始处, zk 类似 zj
  • [z 移动光标到当前打开的折叠的开始处, ]z 类似 [z]

跳转

  • <c-o> 跳转到较新的地方, c-i 跳转到较老的地方

窗口

窗口拆分

  • :sp 上下
  • :vsp 左右

跳转

  • c-w w 在多个窗口直接以此跳转
  • c-w h 向左跳转
  • c-w j 向下跳转
  • c-w k 向上跳转
  • c-w l 向右跳转

调整窗口大小

  • c-w + 增加高度
  • c-w - 减少高度
  • c-w > 增加宽度
  • c-w < 增加宽度
  • c-w = 全部等宽高

移动窗口

  • c-w H 向左移动
  • c-w J 向下移动
  • c-w K 向上移动
  • c-w L 向右移动
